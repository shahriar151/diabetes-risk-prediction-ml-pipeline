# Diabetes Risk Prediction - End-to-End Machine Learning Pipeline

## Overview

This project demonstrates a complete machine learning workflow for diabetes risk prediction using synthetic healthcare data. The pipeline covers data preprocessing, exploratory data analysis, dimensionality reduction, feature selection, model training, evaluation, and deployment through an interactive Streamlit dashboard.

The project was developed for educational and portfolio purposes to showcase practical machine learning, data analysis, and scientific computing skills.
